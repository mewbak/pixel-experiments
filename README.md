# pixel-experiments

These are some of my experiments with the [pixel library](https://github.com/faiface/pixel)

## Installation

Just go get it:

		go get -u -d github.com/peterhellberg/pixel-experiments

## Dependencies

```
github.com/faiface/pixel
github.com/peterhellberg/plasma
```
