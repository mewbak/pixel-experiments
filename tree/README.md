### [tree](/tree/tree.go)

| Key   | Description                |
|-------|----------------------------|
| A     | Increase fraction by 0.005 |
| Z     | Decrease fraction by 0.005 |
| C     | Toggle circles             |
| R     | Red color mask             |
| G     | Green color mask           |
| B     | Blue color mask            |
| W     | White color mask (default) |
| Up    | Increase length by 0.5     |
| Down  | Decrease length by 0.5     |
| Left  | Increase angle by 0.5      |
| Right | Decrease angle by 0.5      |
| 1     | Depth 1                    |
| 2     | Depth 2                    |
| 3     | Depth 3                    |
| 4     | Depth 4                    |
